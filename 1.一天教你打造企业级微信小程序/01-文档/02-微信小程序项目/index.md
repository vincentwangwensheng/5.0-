# 1.bilibili项目介绍

通过原生的微信小程序来实现一个 `bilibili` 应用，来体验原生小程序魅力。

# 2. 技术铺垫

- `html` 和 `css` 和 `JavaScript` 基本功
- 部分 `es6`
- 微信小程序基础

# 3. 关键技术

| ---------- | ------------------------------------------------------------ | -------- |
| 微信小程序 | [开发文档](https://developers.weixin.qq.com/miniprogram/dev/component/) | 官方文档 |
